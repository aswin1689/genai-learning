# GenAI Learning

A hands-on repository for learning **Generative AI Engineering** using
the **Google Gemini API (Python)**.

This repository is designed to teach **how to build AI applications**,
not how to train foundation models from scratch. The focus is on
practical skills used by GenAI Engineers in production.

## Goals

-   Learn how Large Language Models (LLMs) work
-   Master the Gemini Python SDK
-   Build production-ready AI applications
-   Understand Retrieval-Augmented Generation (RAG)
-   Build AI agents and tool-calling workflows
-   Learn deployment, evaluation, and best practices

# Learning Philosophy

-   Understand concepts before frameworks.
-   Build every concept with code.
-   Prefer simple Python over heavy abstractions initially.
-   Learn production practices alongside APIs.
